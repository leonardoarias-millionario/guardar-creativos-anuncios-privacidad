Política de privacidad

Guardar Creativos de Anuncios

Fecha de entrada en vigor: 10 de julio de 2026 Responsable: Leonardo Arias Contacto y soporte: rocketinfoagenda@gmail.com

1. Alcance

Esta política explica cómo la extensión Guardar Creativos de Anuncios trata información cuando el usuario la utiliza en la Biblioteca de Anuncios de Meta o en el Centro de Transparencia Publicitaria de Google.

La extensión tiene un único propósito: permitir que el usuario guarde, mediante un clic explícito, archivos de medios compatibles que ya están visibles en esas bibliotecas públicas, para un uso propio o autorizado.

2. Información tratada

Para prestar su función principal, la extensión procesa localmente y de manera transitoria:

  • La URL de la página actual, únicamente para comprobar que pertenece a uno de los sitios admitidos.
  • La estructura visible de la página necesaria para localizar un creativo.
  • El identificador público del anuncio o creativo, cuando la página lo muestra.
  • El tipo de medio y la URL HTTPS directa de una imagen o un archivo de vídeo compatible.
  • El clic del usuario en el botón de guardado, únicamente como orden inmediata para iniciar la descarga.
  • La confirmación de uso propio o autorizado, la versión del aviso aceptado y la fecha y hora de aceptación.

El contenido público de un creativo puede incluir nombres, imágenes u otra información colocada por el anunciante. La extensión no intenta identificar personas, interpretar ese contenido ni crear perfiles; solo localiza el archivo compatible solicitado por el usuario.

3. Información que no se recopila

La extensión no está diseñada para recopilar:

  • Nombres, direcciones, números de identificación o datos de contacto del usuario.
  • Contraseñas, cookies, tokens o información de autenticación.
  • Información médica, financiera o de pagos.
  • Comunicaciones personales, contactos o contenido de formularios.
  • Ubicación precisa o historial general de navegación.
  • Pulsaciones de teclado, movimientos del ratón o actividad fuera de los botones propios de la extensión.

4. Procesamiento local, transmisión y almacenamiento

El procesamiento se realiza dentro del navegador del usuario. La extensión no dispone de un servidor del desarrollador, no envía los datos descritos al desarrollador y no utiliza servicios de analítica o telemetría.

Los valores necesarios para localizar un creativo existen de forma transitoria durante el análisis de la página y la solicitud de descarga. La extensión no crea una base de datos y no mantiene un historial de anuncios.

La extensión utiliza chrome.storage.local para guardar únicamente la clave de consentimiento authorizedUseConsentV1. Su valor contiene el estado aceptado, la versión authorized-use-v1 y la fecha y hora de aceptación en formato ISO-8601. No contiene nombres, correos, anuncios ni URL de medios y no se sincroniza entre dispositivos. Permanece en el navegador hasta que el usuario pulse Revocar consentimiento, desinstale la extensión o borre sus datos.

Cuando el usuario solicita una descarga, Chrome obtiene el archivo desde la URL de medios de la plataforma y lo guarda en el dispositivo. El archivo descargado queda bajo el control del usuario.

5. Compartición, venta y publicidad

  • No se vende ni se alquila información.
  • La extensión no transmite información al desarrollador ni a terceros adicionales. La solicitud HTTPS de descarga se envía directamente al proveedor que ya aloja el archivo (Meta o Google), únicamente para obtener el archivo solicitado.
  • No se utiliza información para publicidad, marketing, evaluación crediticia o personalización ajena a la función principal.
  • No se muestran anuncios propios.

La conexión necesaria para obtener el archivo se realiza directamente con la infraestructura que ya aloja el medio solicitado. No interviene ningún servidor del desarrollador.

El uso de la información tratada por la extensión, incluida la obtenida mediante las API de Chrome y el contenido de los sitios compatibles, cumple la Política de Datos de Usuario de Chrome Web Store, incluidos sus requisitos de Uso Limitado. La información se utiliza exclusivamente para prestar la función visible y solicitada por el usuario.

6. Permisos y sitios admitidos

La extensión utiliza el permiso downloads para entregar al gestor de descargas de Chrome una descarga solicitada expresamente por el usuario. Utiliza el permiso storage únicamente para guardar localmente la confirmación de uso autorizado descrita en la sección 4.

Los scripts de contenido están limitados a las rutas de la Biblioteca de Anuncios en facebook.com y a adstransparency.google.com. No se solicita acceso a todos los sitios web. En Google, algunos creativos se muestran dentro de marcos internos del mismo sitio; el acceso a esos marcos se utiliza únicamente para localizar imágenes directas compatibles.

7. Límites de la edición de Chrome Web Store

  • Admite imágenes y archivos de vídeo HTTPS directos en la Biblioteca de Anuncios de Meta.
  • Admite imágenes HTTPS directas en el Centro de Transparencia Publicitaria de Google.
  • No descarga vídeos de Google, YouTube o googlevideo.com.
  • No acepta URL blob: o data:.
  • No elude DRM, controles de acceso ni otras medidas técnicas.
  • No reconstruye transmisiones segmentadas ni automatiza recopilaciones masivas.

8. Seguridad

Todo el código ejecutable se incluye en el paquete de la extensión. No se descarga ni ejecuta código remoto. Las URL se validan por protocolo, dominio y plataforma antes de iniciar una descarga. No obstante, ningún software puede garantizar seguridad absoluta; el usuario debe mantener Chrome y la extensión actualizados.

9. Elección y control del usuario

La extensión solo inicia una descarga cuando el usuario pulsa su botón. El usuario puede no pulsarlo, revocar la confirmación de uso autorizado desde el menú, desactivar o desinstalar la extensión y eliminar en cualquier momento los archivos descargados. La revocación elimina el registro local y desactiva la función hasta una nueva aceptación.

Como el desarrollador no recibe ni conserva datos del usuario, normalmente no existe información en sus sistemas que deba consultarse, corregirse o eliminarse.

10. Uso autorizado y servicios de terceros

La extensión es independiente y no está afiliada a Google, Meta o Facebook. El acceso y uso de cada biblioteca también están sujetos a los términos y políticas de su respectiva plataforma.

La disponibilidad pública de un anuncio no transfiere derechos sobre el creativo. El usuario debe utilizar la extensión únicamente con anuncios propios o cuando tenga permiso, licencia u otra base legal válida.

11. Menores de edad

La extensión no está dirigida específicamente a menores ni está diseñada para recopilar información personal de menores.

12. Cambios en esta política

Esta política puede actualizarse si cambian las funciones de la extensión o los requisitos legales. La versión publicada mostrará una nueva fecha de entrada en vigor. Si un cambio modifica de forma material el tratamiento de información, también se actualizarán las declaraciones de privacidad de Chrome Web Store antes de publicar la nueva versión.
